# AAL-116 Brain Connectome

Проект разнесен на понятные части:

- index.html - оболочка страницы.
- src/styles.css - стили интерфейса.
- src/app.js - логика фильтров, графа и экспорта CSV.
- data/connectome.json - редактируемая база модели: узлы, связи, цвета, статистика и словари фильтров.
- endor/plotly-3.3.1.min.js - локальная копия Plotly из исходного HTML.

## Как редактировать модель

Основной файл для правок: data/connectome.json.

- 
odes хранит точки мозга и их свойства.
- edges хранит связи между точками и веса.
- colors задает цвета сетей.
- stats хранит сводные показатели.
- ilters задает списки для боковых фильтров.

После изменения JSON обновите страницу в браузере.

## Как запустить

Из этой папки:

`powershell
python -m http.server 8000
`

Затем открыть http://localhost:8000/.

Важно: из-за загрузки JSON через etch страницу лучше запускать через локальный сервер, а не двойным кликом по HTML.
